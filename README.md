# Rabbit Boy Space Run

Rabbit Boy Space Run is a 2D web platformer built with Vite, Phaser 3, and TypeScript.

## Features

- Keyboard and touch controls
- 2 playable levels
- 3 lives system
- 10 total cosmic parts to collect
- Patrol robots that damage the player
- Rocket unlock after collecting all 10 parts
- Slippery ice physics in Level 2
- HUD for lives and part count

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Vite outputs production files to `dist`.

## Preview Production Build

```bash
npm run preview
```

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`
- Root directory: `/`
